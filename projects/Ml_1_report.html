Predicting the odds of a bank customer accepting a new campaign offer¶

In this study we propose the development of a predictive model according to the Industry Standard Process for Data Mining (CRISP-DM) methodology, to help our client, a Portuguese bank, to maximize the performance of its future telemarketing campaigns. This model was developed using Data Mining (DM) techniques, and allowed us to analyze a set of data, collected between 2008 and 2013 , with a total of 52,944 records. The goal was to prove the relevance of applying a set of machine learning techniques, in order to provide better results in a future telemarketing campaign and if possible with lower costs associated with it.

Classification¶

In the Classification part we tested our machine learning models using the python library skl-learn. We used the ROC curve to avaliate our models, the ROC curve consists of a graph confronting the true positive rate (x-axis) with the false positive rate (y-axis). Thus, the coordinate (0,1) represents a perfect classification, that is, 0 false positives and 0 false negatives. Therefore, the area under this curve is a measure of accuracy, and the curve with a larger area represents the best technique, in the view of this measure. For both classification and regression, when splitting the training and test data, the variables from X were removed:

'17 - y': because it is the target variable of the prototype and classification techniques;

'12 - duration': because this data is only known when the call ends, and the operator needs to know the result of the prediction, in the course of the call.
